
# Easy Suveys

Easy surveys is a survey hosting application. It allows user to create survey from availabe templates and track response analytics.


## Features

- User registartion
- 2 different survey templates
- Survey analytics

## Running the project

Clone this project locally

```bash
  npm install
  npm start
```
Enjoy Easy Surveys!
